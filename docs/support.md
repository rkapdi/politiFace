# Politiface — Support

_Last updated: June 10, 2026_

Quick answers and how to reach us.

## Common questions

### How do I reset my progress?
Settings → Danger zone → Reset progress. This wipes streak, XP, every
review, and onboarding state. Content stays on the device — no need to
re-download.

### Can I sync between devices?
Not yet. Politiface is local-first; your gameplay lives on the device
where you played it. Cross-device sync via an optional account is on
the roadmap.

### A politician's photo is wrong / outdated. How do I report it?
Open an issue at [github.com/rkapdi/politiFace/issues](https://github.com/rkapdi/politiFace/issues)
with the card name and what's wrong. Portraits come from Wikimedia
Commons; we re-fetch on each release.

### The daily round won't start. What do I do?
Try force-quit and reopen. If that doesn't help, Settings → Danger
zone → Reset progress will start you fresh (you'll lose streak data).
If it still doesn't work, email us — see below.

### How do I disable the daily reminder notification?
Settings → Notifications → toggle off Daily review reminder. Or revoke
notification permission entirely in iOS Settings → Notifications →
Politiface.

### Is my data sold or used to train AI?
No. We don't have a server that holds your gameplay. Crash reports
sent to Sentry have personal info stripped. See the [Privacy Policy](privacy-policy.md)
for the full breakdown.

## Reporting bugs

If you can include:

1. What you were doing when it broke
2. The version (Settings → About → Version)
3. Your iPhone model + iOS version
4. A screenshot if visual

…we'll be able to fix it much faster. You can also file a public issue
at [github.com/rkapdi/politiFace/issues](https://github.com/rkapdi/politiFace/issues).
