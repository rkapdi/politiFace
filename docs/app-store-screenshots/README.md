# App Store screenshots

Samples in this folder were captured from a **debug build** for layout
reference. The DEBUG ribbon in the top-right corner means they are
**not** suitable for App Store upload — Apple rejects builds with the
ribbon.

To produce final screenshots, follow the brief in
[`docs/app-store-metadata.md`](../app-store-metadata.md#screenshot-brief-you-capture-these-from-a-release-build).
